  3. What are there three different layer in a system?

    4. Ans. Presentation Layer (UI Interface), Application Layer(Where data is processed), Data Layer(Where data associated with the application)

  4. What is Hyperlink?

    5. Ans. The anchor(a) tag defines a hyperlink, which is used to link from one page to another.

  5. What are entities in HTML?

    6. Ans. An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ).
    Character entities are used to display reserved characters in HTML. A character entity looks like this: &entity_name;. OR. &#entity_number;  

  6. What is difference between button and submit button?

    7. Ans. Submit button is added for a form. When submit is clicked it triggers to the address written in the "action" attribute of form element. Button can be used anywhere as a general purpose. It can be used to redirect to any link and not restricted to a form action.

  7. What are different events for a button?

    • onBlur Event
    • onClick Event
    • onFocus Event
    • onKeyPress Event

  8. What is behaviour of onClick function?

    9. Ans. The onclick event generally occurs when the user clicks on an element. It allows the programmer to execute a JavaScript's function when an element gets clicked. This event can be used for validating a form, warning messages and many more. Using JavaScript, this event can be dynamically added to any element.

  9. What are content(Containers) tags in HTML?
    10. TagDescription
    divA container for a block of content.
    spanA container for in-line content, such as content inside a paragraph.
  10. Use of span tag in HTML.

    11. Ans. The span tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The span tag is much like the div element, but div is a block-level element and span is an inline element.

  11. What is Src attribute in HTML?

    12. Ans. The src(source) attribute specifies the location (URL) of the external resource.

  12. What is href attribute in HTML?

    13. Ans. Href stands for hypertext reference. The href attribute indicates the destination of the hyperlink.

  13. What is rel attribute in HTML?

    14. Ans. The rel attribute specifies the relationship between the current document and the linked document. Only used if the href attribute is present. Tip: Search engines can use this attribute to get more information about a link.
